## Prototype board for doing bus/card/cartridge bringups

Uses two development boards:

* [Waveshare Core2350B](https://www.waveshare.com/core2350b.htm)
* [Freenove ESP32-S3 CAM](https://store.freenove.com/products/fnk0085) dual USB with microSD on the back

The ESP32-S3 board may require soldering to get the microSD card working.
